Display a list of your public beevou vouchers templates, requires an active beevou.net account.

== Description ==

This plugin display a list of your public beevou vouchers templates, it requires an active beevou.net account that Here you hace to enter your beevou that will be used to connect to the beevou.net API with your credentials, also you can specify the data to display, like the voucher icon and description.

== Installation ==

1. Upload the folder `beevou` to the `/wp-content/plugins/` directory or upload the zip through wordpress administrator panel
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your beevou.net account under the settings menu
4. Place [beevou] in any page you want to display the public vouchers templates
